**Description:**

Xync extension for XigmaNAS to manage ZFS replication.

To install this extension, run the following command:
Make sure you are inside the directory you want it to be installed in.
For example, if your are in `/mnt/tank/extensions/xync` it will be installed in
`/mnt/tank/extensions/xync`.

```
fetch --no-verify-peer https://raw.githubusercontent.com/tschettervictor/xigmanas-xync-extension/master/xync-init && chmod +x xync-init && ./xync-init && echo "=> Done!"
```
